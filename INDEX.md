## 🐛 Common Issues & Solutions

| Problem | Solution |
|---------|----------|
| npm install fails | `npm cache clean --force && npm install` |
| Port already in use | `npm run dev -- --port 3000` |
| Modules not found | Delete `node_modules`, run `npm install` |
| Styling issues | Clear browser cache, rebuild with `npm run build` |
| Vite not found | Make sure `node_modules` exists after `npm install` |
